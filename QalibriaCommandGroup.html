    Name Type Description Default

    Parameters:

    Name Type Description Default

    The path to the SNP file.

    Whether to apply VNA correction (True/False).

    enable_use_ratio bool

    Whether to enable 'Use Ratio b/a' (True/False).

    		 False
    enable_switch_term bool

    Whether to enable switch term (True/False).

    		 False

    Type Description
    ProberException

    If the remote command returns an error.
