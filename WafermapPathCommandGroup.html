    WafermapPathCommandGroup

    Bases: CommandGroupBase

    This command group bundles functions for setting up and using the test path of the wafermap.

    A test path defines which dies are tested in which order.

    add_bins(selection)

    Adds dies with specific bin values to the stepping path.

    This method simplifies usage by accepting native Python types instead of custom strings, making it easier to dynamically construct input data.

    Parameters:

    Name Type Description Default
    selection int | list[int] | range | list[range]
    • A single bin number (e.g. 1)
    • A list of bin numbers (e.g. [1, 3, 5])
    • A range object (e.g. range(1, 6)) → "1-5"
    • A list of ranges or mix of int/range (e.g. [range(1, 4), 10])
    		 required

    Returns:

    Name Type Description
    int int

    The resulting path length after adding the dies.

    Raises:

    Type Description
    TypeError

    If the input contains unsupported data types.

    create_from_bin(bin_val)

    Create test path by using all dies with a specific bin.

    Wraps SENTIO's map:path:create_from_bin remote command.

    Parameters:

    Name Type Description Default
    bin_val int | str | PathSelection

    The bin value to use. Can be an int or PathSelection enum.

    		 required

    get_die(seq)

    Get die column and row coordinates from a sequence number.

    Wraps SENTIO's map:path:get_die remote command.

    Parameters:

    Name Type Description Default
    seq int

    The sequence number of the die.

    		 required

    Returns:

    Type Description
    Tuple[int, int]

    A tuple with the column and row coordinates of the die.

    remove_bins(selection)

    Removes dies with specific bin values from the stepping path.

    This method allows more natural Python inputs, avoiding manual string construction.

    Parameters:

    Name Type Description Default
    selection int | list[int] | range | list[range]
    • A single bin number (e.g. 2)
    • A list of bin numbers (e.g. [1, 4, 6])
    • A range (e.g. range(5, 10)) → becomes "5-9"
    • A list of ranges or combination (e.g. [range(1, 4), 6])
    		 required

    Returns:

    Name Type Description
    int int

    Remaining path length after removal.

    Raises:

    Type Description
    TypeError

    If the selection type is not supported.

    select_dies(selection)

    Select dies for testing.

    Wraps SENTIO's map:path:select_dies remote command.

    Parameters:

    Name Type Description Default
    selection TestSelection

    The selection of dies to select.

    		 required

    set_routing(sp, pri)

    Set up path finnding for stepping by specifying a start point position and a row or column priority for routing.

    Wraps SENTIO's map:set_routing remote command.

    Parameters:

    Name Type Description Default
    sp RoutingStartPoint

    The start point of the routing.

    		 required
    pri RoutingPriority

    The priority of the routing (rows first, columns first).

    		 required
