    SiPHCommandGroup

    Bases: CommandGroupBase

    This command group contains functions for working with SiPH applications. You are not meant to instantiate this class directly. Access it via the siph attribute of the SentioProber class.

    move_nanocube_xy(probe, x, y)

    Move NanoCube to the target XY position.

    The movement range is limited to 0 ~ 100 μm.

    Parameters:

    Name Type Description Default
    probe ProbePosition

    The positioner (East or West).

    		 required
    x float

    Target X position (μm), must be in range [0, 100].

    		 required
    y float

    Target Y position (μm), must be in range [0, 100].

    		 required

    Returns:

    Type Description
    tuple[float, float]

    A tuple containing the new X and Y positions after movement.

    move_origin(probe)

    Move SiPH positioner to its origin position.

    The movement includes: - NanoCube XY moves back to 50 μm. - UVW axes move back to the position set during Hover Height training. - If the axis is in "Manual" mode, only NanoCube moves to 50 μm.

    Parameters:

    Name Type Description Default
    probe ProbePosition

    The probe to move to origin position (East or West).

    		 required

    Returns:

    Type Description
    None

    A Response object containing the command execution status.

    set_alignment(probe, fiber_type, coarse, fine, gradient, rotary)

    Set the fast alignment function enable including Coarse, Fine, Gradient, and Rotary/Focal searching.

    Parameters:

    Name Type Description Default
    probe ProbePosition

    The positioner ID to calibrate (East or West).

    		 required
    fiber_type FiberType

    The fiber type ("Single", "Array", "Lensed").

    		 required
    coarse bool

    Enable or disable coarse search (True = ON, False = OFF).

    		 required
    fine bool

    Enable or disable fine search (True = ON, False = OFF).

    		 required
    gradient bool

    Enable or disable gradient search (True = ON, False = OFF).

    		 required
    rotary bool

    Enable or disable rotary/focal search (True = ON, False = OFF, not supported for "Single" fiber type).

    		 required
